# PLOT Mathematica
 A Mathematica package for making beautiful plot.
 
## Usage
```mathematica
<<"PLOT.wl"
```
or use
```mathematica
?function
```
to see the document of the functions.

The code for other examples can be found in PLOT_example.nb.
